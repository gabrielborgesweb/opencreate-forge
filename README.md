# OpenCreate Forge (Protótipo)

OpenCreate Forge é um software de manipulação de imagens. 100% gratuito e de código-aberto.

Este software fará parte de um pacote criativo:

- **OpenCreate Forge:** Manipulador de imagens
- **OpenCreate Scale:** Desenho vetorial
- **OpenCreate Waves:** Criação de músicas (Digital Audio Workstation)
- **OpenCreate Rolls:** Edição de vídeo rápido e profissional

## Por que "OpenCreate"?

O nome foi inspirado em **OpenOffice** da Apache, uma suíte de softwares para escritório.
Mas depois ocorreu um fork (clone do código-fonte) e apoiado por desenvolvedores voluntários até hoje, se chamando de **LibreOffice**.

- **Open:** Software de código aberto
- **Create:** Focado em criação de conteúdo

## Como instalar e executar?

```bash
# Instalar dependencias do projeto
npm i

# Executar o projeto
npm start
```
